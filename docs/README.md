## 기능 목록

- BridgeGenerator
-  [ ] : 0, 1로 무작위 값 생성

- Board
- [ ] : 발판 상태를 담는 객체

- Bridge
- [ ] : 두 개의 List<Board>를 갖는 객체

- BridgeGame
- [ ] : 게임 전체 로직

- GameSettings
- [ ] : 다리 생성

- CrossService
- [ ] : 선택한 발판으로 건널 수 있는지 없는지

- TryService
- [ ] : 시도한 횟수를 카운트

- ContinueService
- [ ] : 게임 종료 / 재시작 조건

- LengthValidator
- [ ] : 빈 문자열인지 확인
- [ ] : 숫자인지 확인
- [ ] : 3이상 20이하인지 확인

- MoveValidator
- [ ] : 빈 문자열인지 확인
- [ ] : U D 인지 확인

- ContinueValidator
- [ ] : 빈 문자열인지 확인
- [ ] : R Q 인지 확인
